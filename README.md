# Привет, я Влад

Fullstack- и product-инженер. Проектирую и собираю веб-продукты целиком: от структуры контента и интерфейса до API, базы данных, CI/CD и production-инфраструктуры.

Мне нравятся понятные системы без лишней сложности: аккуратный код, предсказуемый деплой, доступный интерфейс и документация, с которой можно продолжить работу без отдельной экскурсии по проекту.

## Основной стек

- **Frontend:** Next.js, React, TypeScript, Tailwind CSS, HeroUI, Motion;
- **Backend и CMS:** Node.js, Strapi, WordPress, WooCommerce;
- **Данные:** PostgreSQL, MySQL/MariaDB, Prisma;
- **Delivery:** Docker Compose, Nginx/Caddy, GitHub Actions, VPS;
- **Качество:** ESLint, TypeScript, Playwright, smoke-тесты, SEO и accessibility-аудиты.

## Как я работаю

- начинаю с назначения продукта и реальных пользовательских сценариев;
- держу `main` стабильной, а изменения собираю в коротких тематических ветках;
- автоматизирую проверки и делаю деплой воспроизводимым;
- не храню секреты и production-данные в Git;
- оставляю после проекта понятную карту репозитория и инструкции по запуску.
